Conway's Game of Life

Implemented in Rust using winit, and wgpu.

Conway's game of life is a 2D cellular automata where at each iteration a set of rules decides whether a cell lives / dies / stays alive. Explanation

Troubleshooting:

Requires WebGPU to be enabled. You may need to install the latest version of chrome / safari / firefox to view this.

iOS:

Go to Safari > Settings > enable webgpu > True. This should allow the simulation to run
