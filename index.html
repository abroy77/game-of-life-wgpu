Conway's Game of Life

Click and drag on the grid to paint. Press Play to start the simulation.

Conway's Game of Life is a 2D cellular automaton where simple rules determine whether cells live or die each step. Learn more. Suggestions or feature requests? Open an issue.

Troubleshooting

WebGPU must be enabled. Use a recent version of Chrome, Safari, or Firefox.

iOS

Safari → Settings → Apps → Safari → Advanced → Feature Flags → enable “WebGPU”.

Implementation

Built in Rust using winit and wgpu to leverage your GPU for performance and efficiency.
